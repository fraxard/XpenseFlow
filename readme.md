# XpenseFlow

A modern, lightweight, and privacy-friendly expense tracker built with Vanilla HTML, CSS, and JavaScript.

Version: **v0.68.2**

---

## Features

- Income & Expense tracking
- Real-time balance calculation
- Budget management with progress tracking
- Recurring transactions
- Transaction history with:
  - Search
  - Filters
  - Pagination
- Custom categories
- Receipt image attachments
- Currency selection
- Light & Dark mode
- Responsive UI
- Detailed reports dashboard
- Local storage (no account required)

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API

## Running Locally

```bash
git clone https://github.com/fraxard/XpenseFlow.git
cd XpenseFlow
```

Open `index.html` or run a local server:

```bash
python -m http.server
```

or use the VS Code **Live Server** extension.
